# 🍔 100-burger

**부모와 자녀가 함께 햄버거 100개를 만드는 실시간 협동 게임**

> 두 명이 각각 다른 기기에서 접속해 역할을 나눠 함께 버거를 완성하세요.
> 100개를 달성하면 햄버거 쿠폰이 발급됩니다!

---

## 게임 소개

| 역할 | 담당 |
|------|------|
| 👩 부모 | 게임 방 생성, 초대 코드 공유 |
| 👦 자녀 | 초대 코드 입력 후 참여 |

두 캐릭터가 같은 공간에서 실시간으로 이동하며 재료를 수확하고, 버거를 조립합니다.

---

## 게임 흐름

```
부모가 방 생성 → 초대 코드 공유 → 자녀가 코드 입력 → 함께 플레이
```

### 플레이 방법

1. **농장** 🌱 — 토마토·양상추 씨앗 심기 → 꽃 피면 물 주기 → 수확
2. **운반** 🧺 — 수확한 채소를 들고 냉장고로 이동해 보관
3. **발주대** 📦 — 빵·패티·소스 등 재료 주문
4. **주방** 🍳 — 재료를 조합해 버거 완성
5. **목표** 🎯 — 버거 100개 달성 → 쿠폰 발급 🎉

### 조작법

| 입력 | 동작 |
|------|------|
| 방향키 `↑↓←→` | 캐릭터 이동 |
| `A` 키 | 근처 오브젝트 상호작용 |
| `Esc` | 들고 있는 아이템 내려놓기 |
| 모바일 D-패드 | 터치로 이동 및 상호작용 |

## 기술 스택

| 분류 | 기술 |
|------|------|
| 프론트엔드 | React 18 + Vite |
| 스타일 | Tailwind CSS |
| 상태관리 | Zustand |
| 백엔드/실시간 | Firebase Firestore |

---

## 로컬 실행

```bash
# 1. 의존성 설치
npm install

# 2. Firebase 설정
# src/firebase/config.js 에 본인의 Firebase 프로젝트 정보 입력

# 3. 개발 서버 실행
npm run dev
```

> **Firebase 설정 필요:** Firestore 데이터베이스를 미리 생성해야 합니다.
